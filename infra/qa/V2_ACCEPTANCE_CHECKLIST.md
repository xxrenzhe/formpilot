# FormPilot V2.0 验收清单

> 目的：覆盖 PRD + 全景策略的关键路径，确保 100% 落地。

## A. 账号与鉴权
- [ ] 未登录点击输入框唤起面板，显示登录引导遮罩
- [ ] 邮箱/密码注册成功后可使用生成
- [ ] Google 登录成功后回到 Options 页并生效
- [ ] 退出登录后面板与设置页均显示未登录状态

## B. 核心生成体验
- [ ] 仅生成，不自动填充
- [ ] 输出包含翻译与回复（非中文页面）
- [ ] 一键复制按钮正常工作
- [ ] Cmd/Ctrl + M 唤起面板并自动聚焦补充输入框
- [ ] Enter 快捷复制（有回复时）

## C. 上下文解析
- [ ] 局部上下文 (label/placeholder/nearby) 正常
- [ ] Pro 开启全站上下文，Free 不注入全站
- [ ] 上下文过长时去噪/截断，并在面板显示“已读取上下文 X 段，省略 Y 段”

## D. PII 隐私拦截
- [ ] 密码/邮箱/电话等敏感字段直接阻断
- [ ] 提示“请手动输入”并不发起生成

## E. 长文档模式
- [ ] 自动识别文件上传/设计文档类问题
- [ ] 文档模式显示提示卡片
- [ ] Free 用户长文档触发 Paywall 提示
- [ ] Pro 用户长文档可正常生成

## F. 人设库
- [ ] Free 只能创建 1 个云端人设
- [ ] Pro 最多 5 个云端人设
- [ ] 默认人设切换生效
- [ ] 多设备同步一致

## G. 订阅与支付
- [ ] 超额（第 21 次）触发 403 Paywall
- [ ] Stripe Checkout 跳转成功
- [ ] 支付成功后套餐状态更新为 Pro

## H. 埋点与指标
- [ ] panel_open
- [ ] generate_success
- [ ] copy_success
- [ ] paywall_shown
- [ ] rewrite_click
- [ ] Options 页指标概览正常展示

## I. 技术红线
- [ ] 不篡改宿主 DOM
- [ ] 不模拟按键，不自动填写
- [ ] 所有敏感信息不落前端
